linux常用命令

2020-05-28 / 1 min read
复制文件或文件夹
cp dir1/a.doc dir2 //将dir1下的a.doc文件复制到dir2目录下
cp -r dir1 dir2 //将dir1及dir1下的文件复制到dir2
cp -r dir1/. dir2 //将dir1下的文件复制到dir2
解压缩
tar xf flag.tar.gz
CentOS查看ip
ip addr
将flags文件夹里的文件名输出到txt文本
#ls flags > flags.txt
