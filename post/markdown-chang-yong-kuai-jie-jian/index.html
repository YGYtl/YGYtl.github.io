Markdown常用快捷键

2020-05-24 / 2 min read

##Markdown常用语法

插入代码：(~键3次---也就是三个'`')

Ctrl + Shift + C

插入链接

Ctrl + K

行内代码

Ctrl + Shift + K

无序列表

Ctrl + Shift + L

插入图片

Ctrl + Shift + I

撤销

Ctrl + Z

n阶标题

Ctrl + N

返回Typora顶部

Ctrl + Home

返回Typora底部

Ctrl + End

创建表格

Ctrl + T

创建超链接

Ctrl + K

搜索

Ctrl + F

选中某句话

Ctrl + L

选中某个单词

Ctrl + D

选中相同格式的文字

Ctrl + E

搜索并替换

Ctrl + H

删除线

Alt + Shift + 5

插入图片

Ctrl + Shift + I

公式块

Ctrl + Shift + M

引用

Ctrl + Shift + Q

块元素

转义字符'/'

\   反斜线
`   反引号
*   星号
_   下划线
{}  花括号
[]  方括号
()  小括号
#   井字号
+   加号
-   减号
.   英文句点
!   感叹号

标题级别：#.......

引用文字：> + 空格 + 引用文字

无序列表：使用‘*‘、’+‘、’-’ 都可以(后面要加个空格哦)

有序列表：‘1.’、'2.'等等，后面加个空格就行

任务列表

\-[]不勾选
\-[x]勾选

插入表格(弹出表格设置)

分割线：***或---按回车即得分割线

设置目录

[toc]回车即可

内部链接

[链接](http://example.com/ "标题")
[链接](http://example.net/)

图片

![显示文字](C:\images\cat.png "图片标题")
![显示文字](C:\images\cat.png)
还可以直接拖入

斜体

*单个星号*  或者  _单个下划线_

加粗

**两个星号** 或者  __两个下划线__

删除线

~~删除线~~

下划线

<u>下划线内容</u> 或者 Ctrl + U

表情符号

:smile

下标

<sub>内容</sub>
H<sub>2</sub>O

上标

<sup>内容</sup>
X<sup>2</sup>

高亮

==高亮==

文本居中

<center>重要的内容哦</center>

换行符

</br> 或 空格+空格+Enter
